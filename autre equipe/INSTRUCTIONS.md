# Exécuter le code

## 1. Installer les libs
```bash
pip install pandas numpy xgboost lightgbm catboost scikit-learn scipy networkx
```

## 2. Mettre les données
`train.csv` et `test.csv` dans le même dossier que `solution.py`.

## 3. Lancer
```bash
python solution.py
```

## Résultat
Génère `soumission.csv` (colonnes `id`, `target`).
